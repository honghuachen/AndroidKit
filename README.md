# AndroidKit[![](https://jitpack.io/v/taijuan/AndroidKit.svg)](https://jitpack.io/#taijuan/AndroidKit)
## 添加maven:
    allprojects {
	    repositories {
	        maven { url 'https://jitpack.io' }
	    }
	}
## (PermissionKit)集成
#### module添加引用
	dependencies {
        implementation 'com.github.taijuan.AndroidKit:permissionKit:1.1.2'
	}

	dependencies {

	}
